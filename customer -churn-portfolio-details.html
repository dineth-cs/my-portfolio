Project Overview

ChurnGuard is a sophisticated, enterprise-ready machine learning web application designed to help telecommunications companies proactively identify customer churn. By leveraging historical behavior, contract details, and financial metrics, the application provides actionable insights that allow businesses to implement targeted retention strategies before a customer leaves.

Key Features

Predictive AI Engine

Utilizes a high-performance Random Forest Classifier, trained on a robust dataset of 7,000+ telecommunications profiles. The model evaluates critical features like Contract Type, Tenure, and financial usage to generate real-time churn probability vectors.

Bulk Prediction Engine (Batch Processing)

Designed for scalability, the platform features a bulk processing module. Users can upload large datasets via CSV to perform mass churn analysis, enabling the retention team to prioritize thousands of accounts in seconds.

Financial ROI Calculator

A strategic tool that translates predictive analytics into business value. It allows stakeholders to input retention campaign parameters to estimate potential revenue saved and calculate the Net Return on Investment (ROI) of the retention efforts.

Interactive Analytics Dashboard

Built with Plotly and Streamlit, the dashboard provides a modern "Glassmorphism" UI. It features real-time KPI tracking, churn trend visualizations, and "AI Insights," which transparently visualize how the machine learning model weighs specific features (Feature Importance) to reach its predictions.

Optimized Performance & Privacy

Utilizes model serialization (pickle) and resource caching (@st.cache_resource) to ensure the ML model loads only once, drastically reducing overhead. The application follows a strict privacy-first approach where user inputs are processed in-memory and immediately discarded without being stored.
