Project Overview

MindAura is an advanced mobile personal assistant designed to identify user emotional states through multimodal AI. Operating seamlessly across text, voice, and video inputs, it serves as a comprehensive lifestyle and health companion for proactive mental wellness.

Key Features

Multimodal AI Analysis

Utilizes Convolutional Neural Networks (CNN) for precise facial analysis and Transformer models for accurate text sentiment processing.

Advanced Privacy & Security

Implements AES-256 encryption for data records and a strict "process-and-discard" architecture for all biometric inputs to ensure complete privacy.

Cross-Platform UI

Developed utilizing React Native to deliver a smooth, intuitive mobile experience featuring interactive dashboards and mood calendars.

Gamification & Scalability

Incorporates psychological hooks including achievement badges and a streak system to boost daily engagement. The robust backend architecture supports up to 1,000 concurrent users.
